# Breast Cancer Analysis

## Project Overview
This project focuses on analysing breast cancer data using Python to identify patterns, explore important variables and support data-driven understanding of diagnosis-related factors.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Key Features
- Data cleaning and preprocessing
- Exploratory data analysis
- Visualisation of important trends and patterns
- Analysis of variables related to diagnosis outcomes

## Files Included
- Python analysis notebook
- Dataset
